# Tables

This project utilizes the Spring Framework web to create and delete tables. It also supports basic arithmetic operations within the tables, such as parentheses for grouping, multiplication, division, and subtraction. Additionally, it allows referencing cell coordinates using the format "=A1+B3".

## Installation

1. Clone the repository: `git clone https://github.com/your/repository.git`
2. Navigate to the project directory: `cd project-directory`
3. Install the required dependencies: `npm install`

## Usage

1. Start the application.
2. Open your web browser and navigate to `http://localhost:8080`.
3. Create a new table by clicking on the "Create Table" button.
4. Delete a table by selecting it and clicking on the "Delete" button.
5. Perform arithmetic operations by entering the desired formula in a cell. For example, to add the values from cell A1 and B3, enter `=A1+B3`.
6. The result of the formula will be displayed in the cell.

## Examples

- Creating a Table:
    - Click on the "Create Table" button.
    - Click on the "Create" button to create the table.

- Deleting a Table:
    - Select the table you want to delete.
    - Click on the "Delete" button.

- Performing Arithmetic Operations:
    - Enter the desired formula in a cell.
    - Use parentheses for grouping, '*', '/', and '-' for multiplication, division, and subtraction respectively.
    - For example, to calculate the sum of cells A1 and B3, enter `=A1+B3`.
